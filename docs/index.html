Geostatistics · Optimal transport · GSLIB

Spatial multivariate morphing for confidential simulation workflows.

Point Cloud Morphing implements the Avalos–Ortiz SMMT workflow for independent Gaussian morphing-factor simulation and local multivariate reconstruction.

1. Pair

Use discrete optimal transport to pair empirical target CDFs with independent Gaussian morphing factors.

2. Simulate

Fit direct variograms to mean morphing factors and simulate each factor independently with SGSIM.

3. Map

Use local, spatially selected thin-plate splines in logit space to reconstruct original variables.

Data access

Operational datasets, parameterizations, and generated realizations are confidential. This public repository therefore does not provide a sample dataset, benchmark execution path, or reproduction test.

Software interface

python -m pip install -e ".[dev]"
smmt --help
